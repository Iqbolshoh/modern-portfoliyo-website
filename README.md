# 🚀 Modern Portfolio Website

A modern, interactive, and fully responsive portfolio website built with HTML, CSS, and JavaScript. Ready to deploy on GitHub Pages for free.


## ✨ Features

- 🎨 **Modern Design** - Gradient backgrounds, neon effects, glassmorphism
- ⌨️ **Typing Animation** - Automatically changing text ("Developer, Designer, Creative...")
- 🖼️ **3D Profile Image** - Hover effect with floating animation
- 📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- 📧 **Contact Form** - Accepts messages with frontend validation
- 📰 **Newsletter Subscription** - Email subscription form
- ⬆️ **Back to Top Button** - Appears when scrolling
- 🎯 **Smooth Scrolling** - Smooth navigation between sections
- 🌐 **GitHub Pages Ready** - One-click deployment

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure |
| CSS3 | Styling, animations, gradients |
| JavaScript | Typing effect, forms, scroll |
| GitHub Pages | Free hosting |

## 🚀 How to Deploy on GitHub Pages

### 1. Create a new repository

Go to GitHub.com → New repository → Name it (e.g., `my-portfolio`)

### 2. Upload the files

```bash
git clone https://github.com/Iqbolshoh/modern-portfoliyo-website.git
cd my-portfolio
# Copy index.html file here
git add .
git commit -m "Initial commit"
git push origin main
```

## 3. Enable GitHub Pages
* Go to repository Settings

 * Scroll to Pages section

* Under Branch, select main (or master)

* Click Save
